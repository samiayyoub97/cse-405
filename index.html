    • For loops in PHP are very similar to for loops in c++. The syntax for it includes: for (init counter; test counter; increment counter) { code to be executed; } The init counter initializes the loop counter value. The test counter is evaluated for each loop iteration. If it evaluates to TRUE, then the loop keeps going, but if it evaluates to FALSE, the loop stops. Increment counter means to increase the loop counter value. I created an example of a for loop in PHP code to show how it would be written.

    ?php for ($x = 0; $x <= 10; $x++) { echo "The number is: $x"; } >
    This php code reads the number 0 and sees if it is less than or equal than 10. If it is true then it increaments to the next number which will be 1. It will keep looping until it reaches the end of the statement, which in this case would be 10. For every TRUE statement, it will display "The number is" and it will display the current number.
